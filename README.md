# 23 — 와인 정복 게임 (VINUM IMPERIUM)

와인 지식 퀴즈 × 경영 시뮬레이션 브라우저 게임. `index.html` 더블클릭으로 실행 (설치·서버 불필요).

## 컨셉

로마 또는 조지아 진영을 골라, 포도씨를 들고 고대 세계 10개 지역을 정복한다.
동물의 숲(계절 턴제·아늑함) × 거상(시세·카라반 교역) × 페르시아의 왕자(시간의 모래 퀴즈 타이머).

## v2 — 지도 비주얼 + 캐릭터

- **SVG 고지도**: 지중해~페르시아 양피지풍 지도. 정복 시 진영 깃발 팝 애니메이션 + 영토 글로우, 붉은 펄스 링 = 정복 가능, 자물쇠 = 봉인
- **캐릭터 생성**: 이름 입력 + 초상 선택. 진영별 전통복장 여5/남5 (로마: 스톨라·팔라/토가·월계관, 조지아: 카르툴리 드레스·레차키/초하·파파키) — 전부 인라인 SVG
- **캐릭터 마커**: 지도 위에서 원정·정복에 따라 이동 (이동 애니메이션)
- 구버전 세이브 호환: 이어하기 시 캐릭터 생성만 추가로 진행

## 핵심 루프

1. **개간** — 포도씨 + 골드로 포도밭 조성 (지역당 최대 3개)
2. **재배** — 「다음 계절」로 시간 경과, 봄~가을 성장 (겨울 휴면)
3. **수확 → 양조** — 포도 2 → 와인 1 (양조장 필요)
4. **교역** — 지역별 변동 시세로 판매, 카라반 원거리 교역 (2배 값, 떼일 위험 20%)
5. **원정** — 물자+와인을 들고 가서 **와인 퀴즈 대결** 5문제 중 3문제 정답 시 정복

## 퀴즈

- 지역 티어 = 난이도: WSET L1 → L2 → L3 → Diploma → **Master of Wine** (5단계 × 15문항 = 75문항, 해설 포함)
- **정답 확인하기**: 모르는 문제를 물자(티어×4)로 넘길 수 있음
- 30초 모래시계 타이머 — 시간 초과 시 오답

## 이벤트 (계절마다 확률)

지진(건물 붕괴) · 야만족 침공(퀴즈 방어전, 성벽으로 예방) · 대금 떼임 · 풍년 · 상인 특가 · 음유시인

## 진영 보너스

| 진영 | 보너스 |
|------|--------|
| 조지아 (콜키스) | 수확량 +1, 크베브리 전통으로 판매가 +20% |
| 로마 (라티움) | 건설비 -30%, 성벽 보유 시 전쟁 자동 방어 |

## 기타

- 음악: WebAudio 칩튠 루프 (E단조 8마디), 헤더에서 토글
- 자동 저장: localStorage (`vinum_save`) — 브라우저 껐다 켜도 이어하기
- 승리: 10개 지역 전부 정복
